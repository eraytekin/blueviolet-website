Hakkımda

Merhaba, Ben Eray Tekin. Yazılım Hayatıma 2017 yılında giriş yaptım ve o günden bu güne güncel olarak Web Teknolojileri üzerine çalışmaktayım Yazılıma ilk başladığımda kullandığım diller C#, PYTHON gibi çoğunluklu konsol, web app platformları geliştirmeye yarayan yazılımlardı. Şu an ise kullandığım teknolojilerin çoğu öncelikli olarak Web odaklı. Örnek verecek olursak; HTML | CSS | JS | VUE.JS | REACT.JS ile çalışmalar yapmaktayım. Arayüze olan ilgimi sektöre giriş yaptığım ilk süreçlerde fark edip çok dinamik olmayan web siteler yapmaya başlayarak amatörlüğümü atmaya çalıştım. Güncel olarak göze hitap eden çalışmalarımı elimi günden güne daha da hızlandırıp, daha dinamik yazmak için çalışıyorum. Bunun yanı sıra CSS ve Animasyon bilgimi o alandaki dilleri(SASS/SCSS/LESS) ve frameworkleri(anime.js/swiper.js/bootstrap/tailwind) iyi bir şekilde öğrenip, ileri seviyeye taşıyıp daha dinamik, kullanıcılar tarafından daha göze hitap eden tasarımlar yapmaya devam etmekteyim.
